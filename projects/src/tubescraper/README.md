tubscraper is a scraper for YouTube.

### Javascript Runtimes
Since version 2025.11.12, yt-dlp requires an external Javascript runtime.

The suggested runtime is [deno](deno.com). Install instructions can be found
on the website, or you can run the automated (but insecure) install:
```bash
curl -fsSL https://deno.land/install.sh | sh
```
